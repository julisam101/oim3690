## Elective Web Game: 
    - I started off very basic. I wanted something funny and something that calls one's attention. I used bright, flashing, and neon colors to get the end user interested. I also had the two boxes move around the page that way it takes them a while to click on the buttons. 
    - The hardest part was coming up with the themes. I had 5 games and while I thought about having the same theme across all games, it wasn't ideal. I didn't think it would be fun if it was all the same. So the hardest part was coming up with the game and properly executing. I would have some vision but AI had a different. It was a lot of saving the work, going live, and seeing what exact changes needed to be made. This became hard when implimenting the themes. I had to constently make changes and tell AI what I really wanted. I had to ensure that each theme stayed with its proper game or else. I used lots of conditions. If this happen, then do this. For example, if a player got the answer wrong then it had to reset the game board. But, if they got it right then it would have to automatically proceed to the next round. I had to use lots of API's for this. I had to find a trivia and riddle API that way it would generate different ones for different rounds. From there, I had to ask it to manipulate the API with conditions. I also found it extremely helpful when it came to the math game. After the person got 5 right then it would increase the intensity but if they didn't it would have to decrease the level. 
    - As I was creating the web game, a friend was right besides me giving me their opinions. They suggested the 3D hangmang person which I thought was smart. Initally, AI built a 2D model that didn't really look like a hangman but after sending a few request, it was able to built one. My friend thought about downloading the graduation dipolma as a PDF to make it more personalized. I didn't think AI/github would allow it but it did! It turned out great. Some feedback I got was to reorganize the jeopardy game where you got to choose which topic you would like to do rather than it spitting out random topics. It makes it not only easier but allows more it to be more interactive. It makes the player feel as if they have more than a choice! 
    - Because there's a variety of themes, games, and an option to just play the games one would like, it becomes a lot more interactive than your typically games. The themes are common themes so one isn't left thinking "oh what is this" or "this is random." Lastly, if this is your first time playing there is a surprise at the end which is nice. And if this is your third time playing then you can start saving the grad pdf dipolma and starting collecting them!
